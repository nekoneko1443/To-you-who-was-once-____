## ゲーム概要

#### タイトル：「かつて＿＿だった君へ」

#### ゲームジャンル：シナリオゲーム

#### シナリオゲームについて

とあるシナリオに沿って話を進めていき、ユーザーが選んだ選択肢次第で辿り着くEndが別れるゲームとなっています。  
本ゲームは合計3Endとなっています。

#### あらすじ

主人公はその日、文化祭を控えた高校2年生である。  
いつも通り準備をして学校に向かったはずが、不思議な空間に迷い込むことになった。  
主人公は無事、現実世界へ帰ることができるのだろうか。

#### プレイヤー操作

- ゲームを始める：`python3 main.py`を実行  
  `python3 main.py`を実行すると、スタート画面が表示されます。  
  スタート画面で[Enterキー]を押すことでゲームが始まります。
- ストーリーを進める：[右矢印キー] または [Enterキー]  
  （Enterキーを使う際はメッセージウィンドウに表示が出ます。出ていない時は右矢印キーを使ってください。）
- アイテム選択：[各数字キー]（メッセージウィンドウに表示されます。）  
  ~~（指定数字以外を押すと、ゲームのクラッシュにつながるため、押さないようにお願いいたします。）~~（修正しました。）
- ゲームを終了する：ウィンドウの閉じるボタン
  マウスでウィンドウ左上の閉じるボタン(赤色)を押すと、終了することが可能です。

#### 注意点⚠️

- ストーリーのシナリオの都合上、 **_死亡表現など、精神的にダメージのある表現_** が含まれます。  
  プレイ中に体調が悪くなったり、内容が苦手な方はプレイをお控えください。  
  （ウィンドウの×を押していただくと終了できます。）
- ストーリー進行において、連続で右矢印キーを押下しますと、ゲームがうまく動かなくなる可能性があります。  
   少し間隔を空けた上で押下していただきますようお願いします。  
   （ゲームがうまく動かない際は、1度終了して再起動していただきますようお願いします。）
- **このゲームはフィクションです。**
